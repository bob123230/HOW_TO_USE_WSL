## 我的故事
其实你不难看出，WSL配置记.py讲述的就是我的故事。
一天，我按照视频教程，开启Windows Sandbox。却在开启功能菜单中看到了WSL。
我好奇，就开启了它。我想着他出现在启动菜单中，却没有。
于是，我就查“怎么将WSL添加到启动菜单”，查到了需要装一个发行版。
于是，我安装了。
安装后，却没有桌面。
于是，我想：一定要安装一个桌面！！！
首先尝试了gnome。
```bash
sudo apt install gnome #安装GNOME
gnome-session #启动
```
但是却失败了。但是我看到了失败屏幕。
后来我配置了X11转发，成功了，但是卡顿。
于是，我就尝试了XFCE。
```bash
#注意：此并非正道。
sudo apt install xfce4
xfce4-session
```
成功了，没有X11转发，面板直接出现在了Windows桌面上。
不知如何，我懂了WSL的哲学。但是我还是用XFCE。
一次，我突然发现，旧的方法运行XFCE没用了，于是就融合了。
